# Smoke-чеклист klippa (ручная приёмка)

Полный цикл захват→попап→вставка нельзя автоматизировать вне gnome-shell, поэтому
проверяется вручную после установки и перелогина. Ядро демона покрыто `pytest`
отдельно (см. `docs/README.md` → Тесты).

Подготовка:

```bash
gnome-extensions info klippa@local      # ожидаем State: ACTIVE
systemctl --user status klippad         # ожидаем active (running)
```

Удобно держать открытым лог: `journalctl --user -u klippad -f`.

Вспомогательно — посмотреть историю напрямую (минуя popup):

```bash
busctl --user call org.klippa.Daemon /org/klippa/Daemon \
  org.klippa.Daemon1 GetHistory u 0
```

## Чеклист

- [ ] **Захват текста.** Скопируйте текст в нативном Wayland-приложении (например,
      gedit/Text Editor) → запись появляется в истории (`GetHistory` или popup).
- [ ] **Захват из XWayland.** Скопируйте текст в X11-приложении → тоже захвачено.
- [ ] **Дедупликация/MRU.** Скопируйте уже имеющееся значение повторно → дубля нет,
      запись поднимается в начало списка.
- [ ] **Пропуск секретов.** Скопируйте пароль из KeePassXC/менеджера паролей →
      запись **не** появляется в истории.
- [ ] **Захват изображения.** Скопируйте картинку (например, из браузера: «Копировать
      изображение») → в истории запись-картинка с превью размеров.
- [ ] **Скриншот.** Сделайте скриншот в буфер (PrtSc / Shift+PrtSc по настройкам
      GNOME) → автоматически попадает в историю как изображение.
- [ ] **Вызов меню у курсора.** Нажмите хоткей (по умолчанию `<Super>v`) → popup
      открывается рядом с курсором, не уезжая за край экрана.
- [ ] **Навигация клавиатурой.** Стрелки ↑/↓ двигают выделение; мышь не нужна.
- [ ] **Фильтр.** Начните печатать → список сужается по подстроке; Backspace
      возвращает; в подсказке виден текущий запрос.
- [ ] **Миниатюры.** Для записей-картинок в меню видна миниатюра.
- [ ] **Выбор + авто-вставка.** Поставьте курсор в текстовое поле, вызовите меню,
      выберите запись, Enter → значение вставляется в это поле (как Win+V).
- [ ] **Только в буфер.** Поставьте `auto_paste = false` в конфиге → после Enter
      значение лишь в буфере, авто-вставки нет (вставьте `Ctrl+V` сами).
- [ ] **Удаление записи.** В открытом меню наведите выделение, нажмите `Delete` →
      запись исчезает из истории.
- [ ] **Esc.** Закрывает меню без выбора.
- [ ] **Живое обновление.** При открытом меню скопируйте новое значение → список
      обновляется без переоткрытия.
- [ ] **Лимит истории.** Установите `max_entries = 5`, скопируйте >5 значений →
      хранится ровно 5, старейшие вытесняются.
- [ ] **Hot-reload конфига.** Смените `hotkey` в `~/.config/klippa/config.toml` →
      новый хоткей работает без перезапуска.
- [ ] **Персистентность.** `systemctl --user restart klippad` → история сохраняется
      (читается из зашифрованной БД).
- [ ] **Очистка.** `busctl --user call org.klippa.Daemon /org/klippa/Daemon
      org.klippa.Daemon1 Clear` → история пуста и остаётся пустой после рестарта.
- [ ] **Безопасность на диске.** `strings ~/.local/share/klippa/history.db | grep -i
      <ваш-текст>` → искомое **не** находится (зашифровано). Права файла — `0600`.
