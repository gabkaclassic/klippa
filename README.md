# klippa

Минималистичный менеджер буфера обмена для **GNOME Shell / Wayland** (Ubuntu).

Гибрид: тонкое GNOME-расширение ловит буфер и рисует меню у курсора, а Python-демон
держит историю, конфиг и шифрованное хранилище. Подробности, установка, архитектура
и безопасность — в [`docs/README.md`](docs/README.md).

```
extension (GJS)  ──D-Bus──▶  klippad (Python)
захват + popup              история + крипто + конфиг
```
